Task 3: Make it interactive! Introduction to Javascript

So as you've learned so far, HTML and CSS display content of a webpage in a pretty, yet static way.

"Static? What are you even talking about?" What if you want your website to respond to user events such as a mouse click, or the typing of a key? That's where Javascript comes in to make your page interactive!

Here's what a simple HTML page looks like.


<!doctype html>
    <head>
        <link href="style.css" rel="stylesheet">
    </head>
    <body>
        ...
        <script src="app.js"></script>
    </body>
</html>

To include CSS files in HTML we used the link element tag as we've learned before.

Similarly, to include JavaScript files in HTML, we use the script element tag.

The <script> element tells the browser where to find a JavaScript file. It has an attribute inside the opening <script> tag named src, which has the address of the JavaScript file.

Let's see a few examples

Buttons and clicking actions

Pretty neat huh? This is what we call interactive webpages, where they respond to user events such as clicking with a mouse.

To create a button such as this we first need to add some HTML code to actually generate the button.

As you may have guessed it there's a HTML tag to create buttons! The <button> tag.

<button>Click Me!</button>

Go ahead and try adding a button tag to your HTML page and try to see it on the browser.

Oh? What happened? The button itself is created but it's ugly and it doesn't respond to the user interaction, such as a mouse click. Well this is to be expected because as you've learned before to make it prettier, or at least as pretty as the one you saw above you need CSS, in this case we'll just add a class to our button.

<button class="dropbtn">Click Me!</button>

After that we need to write some CSS code to work with that class we just defined for our button to make it style. Go ahead and write some CSS of your own to make the button pretty or if you want it to look like the one above simply use this code.


/* Dropdown Button */
.dropbtn {
    background-color: #4CAF50;
    color: white;
    padding: 16px;
    font-size: 16px;
    margin-left: 10px;
    margin-top:10px;
    border: none;
    cursor: pointer;
}

.dropbtnON {
    background-color: #4CAF50;
    color: white;
    padding: 16px;
    font-size: 16px;
    margin-left: 10px;
    margin-top:10px;
    border: none;
    cursor: pointer;
}

.dropbtnOFF {
    background-color: #ff1000;
    color: white;
    padding: 16px;
    font-size: 16px;
    margin-left: 10px;
    margin-top:10px;
    border: none;
    cursor: pointer;
}

/* Dropdown button on hover & focus */
.dropbtn:hover, .dropbtn:focus {
    background-color: #3e8e41;
}

Okay our button is coming along nicely, it's pretty and styled but it still doesn't respond to mouse clicks. To make it respond to mouse clicks you must first understand what events are.

HTML events are "things" that happen to HTML elements. When JavaScript is used in HTML pages, JavaScript can "react" on these events.

An HTML event can be something the browser does, or something a user does. Here are some examples of HTML events:

  • An HTML web page has finished loading
  • An HTML input field was changed (eg: a user wrote something in a text field)
  • An HTML button was clicked

Often, when events happen, you may want to do something. JavaScript lets you execute code when events are detected.

HTML allows event handler attributes, with JavaScript code, to be added to HTML elements.

<button some-event="some JavaScript code here">

HTML allows event handler attributes, with JavaScript code, to be added to HTML elements.

In the following example, an onclick (user clicks on the HTML element) attribute (with code), is added to a button element:

<button onclick='this.textContent=Date()'>The time is?</button>

What we did there is add some Javascript code to the button, so that it "reacts" when it's clicked upon, and calls a function called Date which is already defined by Javascript and all it does is give you the value of the current time and date when it was called, and sets the button content to that date.

JavaScript code is often several lines long. It is more common to see event attributes calling functions that are defined in separate Javascript files or <script> tags:

In the following example, we already defined the function displayDate in a <script> tag above like this:


<script>
function displayDate() {
    document.getElementById("demo").innerHTML = Date();
}
</script>

So now we create a button and set its onclick event to the function displayDate so our HTML page looks like this:


<!DOCTYPE html>
    <html>
    <body>

        <script>
            function displayDate() {
                document.getElementById("demo").innerHTML = Date();
            }
        </script>

        <p>Click the button to display the date.</p>

        <button onclick="displayDate()">The time is?</button>

        <p id="demo"></p>

    </body>
</html>

So what we're actually doing here is creating a button that when you click on it, it reacts by calling the Javascript method Date and putting the current date that's returned in the paragraph that has the attribute id. Go ahead and try it yourself! Copy the HTML above or come up with your own idea of how to make a onclick interaction of a button!

Playing around with videos

Okay now that we've learned how to make our page react to a click event, let's try something more fun. How about playing/pausing a video? Maybe make it bigger or smaller? Does that sound cool to you? It sure is! Let's see how we can go about it.





So in HTML we have a <video> tag that allows us to display video in our webpages, pretty cool huh? Let's go ahead and add one to our page.


<video id="myVideo" width="420">
    <source src="http://www.quirksmode.org/html5/videos/big_buck_bunny.mp4" type="video/mp4">
</video>

If you add the code above to your HTML page you'll see that a video now appears in your webpage, although it really looks like a still image because you there are no easy ways to play and pause the video! Let's add some with the knowledge we just got about buttons and on click events!


<button onclick="playPause()">Play/Pause</button>
<button onclick="makeBig()">Big</button>
<button onclick="makeSmall()">Small</button>
<button onclick="makeNormal()">Normal</button>

<script>
    var myVideo = document.getElementById("video1");

    function playPause() {
        if (myVideo.paused)
            myVideo.play();
        else
            myVideo.pause();
    }

    function makeBig() {
        myVideo.width = 560;
    }

    function makeSmall() {
        myVideo.width = 320;
    }

    function makeNormal() {
        myVideo.width = 420;
    }
</script>

Okay that was a lot take in, so let's break it down so you can better understand what we did there!

First we added 4 buttons and to each button we assigned a function to its onclick event, then below we added a <script> tag and inside it we defined a variable that points to our video defined in the HTML by its id attribute and then each of our javascript functions that we assigned to the buttons! Their names are self-explanatory but basically the function playPause() makes the video play and pause depending on its current state and the functions makeBig,makeSmall and makeNormal resize the video to a big, small and normal size respectively.

Ok so far so good! Our HTML now should look similar to this:


<!DOCTYPE html>
<html>
<body>

    <div style="text-align:center">
      <button onclick="playPause()">Play/Pause</button>
      <button onclick="makeBig()">Big</button>
      <button onclick="makeSmall()">Small</button>
      <button onclick="makeNormal()">Normal</button>
      <br><br>
      <video id="myVideo" width="420">
        <source src="http://www.quirksmode.org/html5/videos/big_buck_bunny.mp4" type="video/mp4">
      </video>
    </div>

    <script>
    var myVideo = document.getElementById("video1");

    function playPause() {
        if (myVideo.paused)
            myVideo.play();
        else
            myVideo.pause();
    }

    function makeBig() {
        myVideo.width = 560;
    }

    function makeSmall() {
        myVideo.width = 320;
    }

    function makeNormal() {
        myVideo.width = 420;
    }
    </script>

</body>
</html>

Pretty cool stuff huh? Now you've learned how to add multimedia content to your webpage and actually control it with user interactions! If you'd like to know more about multimedia in HTML check the resources below!

Lists: Adding and Removing

We learned how to interact with our webpage and even how to add multimedia content to it! Now let's learn one last cool trick.

One other cool HTML element is the list, and there are some pretty neat things we can do with lists! Let's see an example:

Choose a name:

Input something on the text field above and then click on the button to add the name you input at the end of the list.



Whoa! That was cool! But how exactly can you make your webpage do something like that?

Okay so there's actually quite a few steps to make it work like above so let's break it down.

First we need to actually create some sort of list-like element that will hold our items, and populate it with something so that it doesn't look all empty.


<form>
    <select id="mySelect" size="10">
        <option>Apples</option>
        <option>Pears</option>
        <option>Bananas</option>
        <option>Oranges</option>
    </select>
</form>

This creates a list-like element and populates it with some pre-defined elements. Next we need some sort of input field so that the user can input data to create a new item for the list.


Choose a name: <input id="inputField" type="text" name="inputName"><br>

This creates an input field for text where the user can type any name to add to list.

After this we need to create the buttons that will work all the magic and pick up the information the user inputs.


<button onclick="addElement()">Insert</button>
<button onclick="removeElement()">Remove selected item</button>

Note that we defined the onclick events for the buttons and pointed them to functions, but yet we didnt define those functions on our page, would you be able to do it on your own? You have to use a... you guessed it, a <script> tag.


<script>
    function addElement() {
        var x = document.getElementById("mySelect");
        var y = document.getElementById("inputField");
        if(y.value !== "")
        {
            var option = document.createElement("option");
            option.text = y.value;
            x.add(option);
            y.value = "";
        }
    }
    
    function removeElement(){
        var x = document.getElementById("mySelect");
        if(x.selectedIndex >= 0)
        {
            x.remove(x.selectedIndex);
        }
    }
</script>

Again that was quite a lot, so let's try to understand it bit by bit. First we define a function addElement() that will retrieve the value of the input field and if it's not empty it will then create a new option for the list with the name that was input.

After that we define the function that removes the selected item from the list removeElement() that will check to see if there's actually a selected item on the list and if there is then it removes it from it.

Okay that was a lot to take in right? But if you stuck it out until now your HTML should look similar to this:


<!DOCTYPE html>
<html>
    <body>
    
        <form>
            <select id="mySelect" size="10">
                <option>Apples</option>
                <option>Pears</option>
                <option>Bananas</option>
                <option>Oranges</option>
            </select>
        </form>
        <br>
        
        Choose a name: <input id="inputField" type="text" name="inputName"><br>
        
        <p>Input something on the text field above and then click on the button
            to add the name you input at the end of the list.</p>
        
        <button onclick="addElement()">Insert</button>
        <button onclick="removeElement()">Remove selected item</button>
        
        <script>
            function addElement() {
                var x = document.getElementById("mySelect");
                var y = document.getElementById("inputField");
                if(y.value !== "")
                {
                    var option = document.createElement("option");
                    option.text = y.value;
                    x.add(option);
                    y.value = "";
                }
            }
        
            function removeElement(){
                var x = document.getElementById("mySelect");
                if(x.selectedIndex >= 0)
                {
                    x.remove(x.selectedIndex);
                }
            }
        </script>
    
    </body>
</html>
