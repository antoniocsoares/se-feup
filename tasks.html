Tasks

So what will you do ?

During this next months you will learn how to create a webpage, your first webpage !

It may sound scary and difficult for someone that has never done it before...
Just let us guide you into the world wide web !

This webquest will help you from creating your first lines of code to have a functional and pretty webpage for whatever purpose you may want

You will have to code but don't worry everything will be explained to during the different activities.

Do you need something ?

Before beginning this webquest and even if every software you will need will be explained later here is a brief overview:

  1. A browser: make sure to have a browser that supports the latest technologies.
    if you are reading this you already have one ;)
  2. A notepad: you are going to need to type texts (code in our case) so make sure to have a software that allows that:
    • If don't have one already installed by default like notepad on windows or textedit on mac you can download a free one here : Notepad++

What will you learn ?

As this is a very important point here is what you can considered to have learn at the end of this webquest:

  • You will be able to create HTML code and understand what it is and how it is used
  • You will be able to create CSS code and understand what it is and how it is used
  • You will be able to create Javascript code and understand what it is and how it is used
  • You will be able to used everyday software to create your own web pages

Time to learn !

So by now you are ready to start your journey to the web pages creation!

Everything you will need to know is going to be explained during the quest but if you have any doubts the help is here for you !
