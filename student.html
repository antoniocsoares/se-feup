Student Help

In this page you'll find some basic information about HTML, CSS and Javascript that might be of help to you in case you get stuck somewhere along the tasks.

What's the simplest HTML page template?

The code below should be the only thing you need to start making your webpage.


<!DOCTYPE html>
<html>
     <head>

    </head>
    <body>

    </body>
</html>

How do I insert a paragraph in HTML?

To insert a paragraph in a webpage you need to use the <p> tag


<p>
    Insert your text here
</p>

How do I add a CSS file to my HTML page?

To reference a CSS file you created in your HTML page use the following code


<link rel="stylesheet" type="text/css" href="path-to-your-css-file"></link>

How do I add a Javascript file to my HTML page?

To reference a Javascript file you created in your HTML page use the following code


<script src="path-to-your-javascript-file"></script>
