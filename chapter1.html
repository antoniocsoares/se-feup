HTML and CSS

To create a web site we need to provide to the computer instructions, it's not enough just to type texts that should appear on the website (like one would do on word for instance) you also have to tell the computer where to put the text insert images make links between pages and so on

The roles of html and css

To explain to the computer what you want to do you will need to use a language he understands.

That's where things get tricky as you will have to learn two languages

Why two languages ? One would be enough right ? 

You must be asking that to learn and understand two languages will be double the complexity and twice as much time to learn… but it's not true !

Those two languages are here to facilitate things because they complete each other:

HTML (hypertext markup language): was created back in 1991 when the web was launched. His role is to organise the content. It's therefore in HTML that you will write what needs to appear on the page: texts, links, images…

CSS (Cascading Style Sheets): the role of CSS is to handle the appearance of the web page (position, style, colours, etc…) that language came up in 1996 to complete the HTML

Info: you may have also heard of the XHTML, it's a variation of the HTML that is more rigourous and can be a bit more complicated to handle, in our course we will work on the latest version of HTML (HTML 5) which is the language that today everybody is inclined to use

You can create websites only using HTML but it won't be very pretty that's why you need to use CSS to complete your website.
